Open-eCH EWK Webstart

Mit einem Klick auf den Launch-Button laden und starten Sie die aktuelleste Version der Einwohnerkontrolle von Open-eCH. Alle benötigten Bestandteile werden automatisch installiert und konfiguriert. Sie können Open-eCH EWK direkt nach der Installation benutzen.

Version 0.8.91, 24.3.2011

Hinweis: Open-eCH EWK Webstart benötigt Java 6.

Open-eCH Core Demo

Dieses Beispielprojekt soll aufzeigen, wie die wichtigsten Funktionen der Open-Ech Core Klassen für eigene Projekte verwendet werden können.

Zu einer einfachen Bücherverwaltung wird eine Datenbank definiert, eine Eingabemaske sowie eine Suchabfrage erstellt. Die Masken enthalten dabei Codefelder, Checkbox-Felder, Datumsfelder sowie eine Validierung basierend auf der Datenbank. Die Fensterverwaltung geschieht in der Art eines Browsers und behandelt die Rückfragen bei ungespeicherten Eingaben.

Hinweis:
Die Zip-Datei enthält ein Eclipse Projekt und ist für Programmierer gedacht. Als Show Case ist Open-eCH EWK besser geeignet.
