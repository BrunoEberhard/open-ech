Technik

Open-eCH soll auch immer wieder Möglichkeiten und Grenzen neuer und innovativer Software Entwicklung aufzeigen. Ein zentrales Thema ist dabei die Reduktion oder Begrenzung von Komplexität. Komplexität ist der grösste "Killer" bei Software Projekten.

2012: Web / Cloud

Die Zeit der rein lokal installierten Anwendung ist lange vorbei. Die verwendeten Techniken die als "Standard" gelten wechseln sich nur allzuhäufig ab. Daher hat Open-eCH in 2012 versucht, Technik und Fachlichkeit in der Ebene der Implementierung viel stärker zu trennen, als bei gewöhnlichen Projekten der Fall ist. Herausgekommen ist eine Methodik, die eine Beschreibung des Buisness erlaubt, die dann in verschiedenen GUI Techniken verwendet werden kann. Zur Zeit werden eine Web - Implementierung und eine Desktop - Implementierung unterstützt.

Die Web - Variante der Software ist bei dem Cloud Anbieter "CloundFoundry" installiert und kann dort getestet werden. Die Adresse lautet http://openech.cloudfoundry.com.

2011: Pure Java

Die erste Version der Software bestand praktisch aus reiner Java Technologie ohne weitere Zusätze zu dem JDK. Damit konnte eine sehr schlanke Applikation von unter 5 MB erstellt werden. Allerdings beschränkte sich die Software auf eine Einzelplatz Installation. Diese Version wird zur Zeit nicht mehr weiterentwickelt, kann hier jedoch noch mit Webstart gestartet werden.

Version 0.8.91, 24.3.2011

Hinweis: Open-eCH EWK Webstart benötigt Java 6.

2011: Datenbank

Die Daten werden transparent historisiert in einer eigenen SQL-basierten Datenbank abgelegt. Es ist zu einem späteren Zeitpunkt problemlos möglich die Daten in einer sich bereits im produktiven Einsatz befindlichen Datenbank (z.B. MS SQL-Server, Oracle DB) zu verwalten.
